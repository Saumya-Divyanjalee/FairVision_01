 # FairVision  
### CNN-Based Age Group Classification with Bias Detection & Mitigation  

**Saumya Divanjalee | IJSE  

---

## Overview

FairVision is a deep learning computer vision project that performs **age group classification** using a CNN model trained on the **FairFace dataset**.

The project focuses on identifying and reducing **demographic bias (race and gender)** using two mitigation techniques:

- M1: Class-Weighted Loss Function  
- M2: WeightedRandomSampler (Balanced Batch Sampling)

The system compares baseline performance with fairness-enhanced models.

---

## Key Features

- CNN-based age classification (9 classes)
- Bias detection across race and gender
- Two fairness mitigation strategies
- Performance and fairness evaluation
- Streamlit web application for real-time prediction
